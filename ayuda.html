Introducción al Sistema de Control de Partidas

Bienvenido a la sección de ayuda de nuestro sistema. Aquí encontrarás información detallada sobre cómo utilizar las diferentes funcionalidades y módulos para gestionar tus partidas presupuestarias de manera eficiente.

Este sistema está diseñado para simplificar el seguimiento, la asignación y el control de los recursos económicos, asegurando una gestión transparente y organizada.

Gestión de Partidas Presupuestarias

Este módulo te permite crear, visualizar, editar y eliminar las partidas presupuestarias de tu organización.

Crear una Nueva Partida

Para añadir una nueva partida al sistema:

  1. Navega a la pestaña "Partidas" o "Gestión Presupuestaria" en el menú principal.
  2. Haz clic en el botón Añadir Nueva Partida.
  3. Se abrirá un formulario. Rellena los siguientes campos:
    • Nombre de la Partida: Un título claro y conciso para la partida (ej. "Gastos de Oficina", "Salarios", "Inversión en Tecnología").
    • Código de Identificación: Un código único para identificar la partida (opcional, si aplica a tu sistema).
    • Monto Asignado: El valor presupuestado para esta partida. Utiliza solo números y el separador decimal adecuado.
    • Fecha de Inicio: La fecha en que la partida entra en vigor.
    • Fecha de Finalización: La fecha en que se espera finalizar el uso de esta partida.
    • Descripción: Una breve explicación del propósito o uso de la partida.
    • Estado: Selecciona el estado actual de la partida (ej. "Activa", "Inactiva", "Cerrada").
  4. Una vez completados los campos, haz clic en el botón Guardar Partida.
  5. Recibirás una notificación ¡Partida guardada exitosamente! si la operación fue correcta.

Editar una Partida Existente

Para modificar la información de una partida ya creada:

  1. En la tabla de listado de partidas, localiza la partida que deseas editar.
  2. En la columna "Acciones", haz clic en el icono (Editar).
  3. El formulario de la partida se precargará con los datos actuales. Realiza los cambios necesarios.
  4. Haz clic en el botón Actualizar Partida para guardar los cambios.

Eliminar una Partida

Para eliminar una partida de forma permanente:

  1. En la tabla de listado de partidas, localiza la partida que deseas eliminar.
  2. En la columna "Acciones", haz clic en el icono (Eliminar).
  3. Se te pedirá confirmación antes de eliminar. Asegúrate de que es la partida correcta, ya que esta acción es irreversible.
  4. Confirma la eliminación.

Gestión de Usuarios y Roles

Este módulo permite administrar los usuarios que tienen acceso al sistema y asignarles roles para definir sus permisos.

Añadir un Nuevo Usuario

  1. Dirígete a la pestaña "Administración" y luego a la subsección "Gestión de Usuarios".
  2. Haz clic en el botón Añadir Usuario.
  3. Completa el formulario con la información del nuevo usuario:
    • Nombre Completo: El nombre y apellido del usuario.
    • Correo Electrónico: Será el nombre de usuario para iniciar sesión.
    • Contraseña: Establece una contraseña segura para el usuario.
    • Confirmar Contraseña: Repite la contraseña para verificar.
    • Rol: Selecciona el rol que tendrá el usuario (ej. "Administrador", "Contador", "Visualizador").
  4. Haz clic en Crear Usuario.

Gestionar Roles y Permisos

Los roles determinan qué acciones puede realizar un usuario en el sistema.

  1. En la pestaña "Administración", selecciona la subsección "Gestión de Roles".
  2. Para añadir un nuevo rol, escribe su nombre (ej. "Editor de Partidas", "Auditor") en el campo de texto y haz clic en el botón Añadir Rol.
  3. La tabla de roles mostrará todos los roles existentes. Desde aquí, puedes Eliminar roles que ya no sean necesarios.

Reportes y Análisis

El módulo de reportes te ofrece una visión general y detallada del estado de tus partidas presupuestarias mediante gráficos interactivos.

Filtrar y Visualizar Datos

En la sección de "Reportes", encontrarás opciones para filtrar los datos por:

  • Rango de Fechas: Para ver el desempeño en un periodo específico.
  • Tipo de Partida: Si las partidas están categorizadas.
  • Estado: Para analizar partidas activas, cerradas, etc.

Los gráficos se actualizarán dinámicamente con los filtros aplicados. Puedes cambiar el tipo de gráfico (barras, líneas, pastel) si la funcionalidad está disponible.

Exportar Informes

Puedes exportar los datos visualizados en los reportes a diferentes formatos:

  • Exportar a Excel: Genera un archivo `.xlsx` con los datos de la tabla o del reporte.
  • Exportar a PDF: Crea un documento PDF imprimible del reporte actual.
