¿Qué es SIGESCON?

SIGESCON es una aplicación web desarrollada en HTML + JavaScript (con Dexie.js, Chart.js y Bootstrap) diseñada para gestionar contratos empresariales, Hojas de Entrada de Servicios (HES), avances físicos, informes y estadísticas.

  • Tecnología: Frontend puro (sin backend)
  • Almacenamiento: Base de datos local con Dexie.js (IndexedDB)
  • Exportaciones: Excel y PDF
  • Gráficos: Chart.js

Barra de Navegación

La barra lateral contiene todas las funcionalidades del sistema, divididas en módulos independientes:

  • Resumen General: Muestra estadísticas clave del sistema.
  • Nuevo/Editar Contrato: Formulario para crear o actualizar contratos.
  • Lista de Contratos: Consulta, filtra y exporta tus contratos.
  • Gestión de HES: Registra Hojas de Entrada de Servicios asociadas a contratos.
  • Avance Físico: Registro manual de avances físicos por contrato.
  • Resumen Gráfico: Visualización gráfica de los datos.
  • Informes: Genera informes completos sobre contratos y HES.

Módulo: Nuevo/Editar Contrato

Este formulario permite registrar nuevos contratos o editar uno ya existente. Contiene campos como:

  • Número de Proveedor
  • Fecha de Firma del Contrato
  • Fecha de Creación
  • Fecha de Inicio y Fin
  • Duración (calculada automáticamente)
  • Monto Original, Modificado y Total
  • Modalidad de Contratación (dinámica con botones + y -)
  • Partidas (agregables dinámicamente)
  • Archivos adjuntos

Añadir Partida

Haz clic en "Añadir Partida" para agregar filas con información de partidas contractuales.

// Ejemplo de partida
[
  { description: "Instalación eléctrica", quantity: 10, umd: "Unid", unitPrice: 150.00 },
  { description: "Pruebas técnicas", quantity: 5, umd: "Horas", unitPrice: 75.00 }
]

Guardar/Actualizar

Guarda el contrato actual y actualiza automáticamente el avance financiero global.

Módulo: Lista de Contratos

Muestra todos los contratos guardados, permitiendo aplicar filtros por:

  • Número de Proveedor
  • Número SICAC
  • Rango de Fechas
  • Modalidad de Contratación
  • Estatus (Activo, Finalizado, etc.)

Incluye dos botones principales:

Aplicar Filtros

Filtra los resultados según los criterios seleccionados.

Limpiar Filtros

Restablece todos los campos de filtro y recarga toda la lista.

Módulo: Gestión de HES

HES significa Hojas de Entrada de Servicios. Este módulo permite:

  • Seleccionar un contrato
  • Agregar nuevas HES
  • Asociar partidas a cada HES
  • Calcular automáticamente el subtotal, gastos administrativos y total

Cuando se crea una HES, se registran automáticamente:

  • Porcentaje de avance físico por partida
  • Porcentaje de avance financiero

Tabla de Partidas Asociadas

Cada fila muestra la descripción, cantidad original, UMD, precio unitario y cantidad ejecutada.

Módulo: Avance Físico

Permite registrar avances manuales de progreso físico en porcentajes (%) para cada contrato.

Registrar Avance

  • Elige un contrato
  • Ingresa la fecha y el porcentaje de avance
  • Escribe una descripción opcional
  • Haz clic en "Registrar Avance"

Historial de Avances

Se guarda un historial completo de avances físicos por contrato, mostrando:

  • Fecha
  • Porcentaje
  • Monto del avance
  • Descripción

Módulo: Resumen Gráfico

Ofrece representaciones visuales de los datos almacenados, basadas en las modalidades de contratación.

Tipos de Gráficos Soportados

  • Barras: Comparación visual entre modalidades
  • Torta: Distribución proporcional
  • Rosquilla: Porcentajes y distribuciones

Exportar Gráfico

Puedes descargar el gráfico en formato:

  • Excel: Datos usados en el gráfico
  • PDF: Imagen del gráfico
  • PNG: Descarga directa del gráfico

Módulo: Informes

Genera reportes completos de cualquier contrato seleccionado, incluyendo:

  • Detalles del contrato
  • Partidas asociadas
  • Historial de HES
  • Gráficos interactivos de avance

Ver Detalles

Al hacer clic en el ícono de detalles, puedes ver la estructura interna del contrato desde la consola del navegador.

Exportaciones Disponibles

SIGESCON permite exportar datos a múltiples formatos para compartirlos fácilmente.

Exportar a Excel

Descarga una hoja de cálculo con todos los contratos o datos del gráfico.

Exportar a PDF

Obtén un documento PDF con tablas de datos ordenadas y fáciles de compartir.

Funcionamiento de Botones

Los botones están organizados por funcionalidad y tienen comportamientos específicos:

  • Botón de Ayuda: Redirige a este mismo documento.
  • Botón de Exportar: Permite guardar datos en Excel o PDF.
  • Botón de Actualizar: Recalcula avances y totales tras cambios.
  • Botón de Eliminar: Elimina registros permanentemente (con confirmación).
  • Botón de Editar: Carga los datos previos para modificarlos.

Notificaciones (Toast)

SIGESCON utiliza notificaciones emergentes tipo toast para mostrar mensajes al usuario:

  • Éxito: ✔️ Acción realizada correctamente
  • Error: ❌ Ha ocurrido un problema
  • Advertencia: ⚠️ Algo necesita atención

Estas notificaciones aparecen durante 3 segundos y ofrecen retroalimentación inmediata al usuario.

Manejo de Archivos Adjuntos

SIGESCON permite adjuntar archivos tanto a contratos como a HES. El sistema maneja los archivos de la siguiente manera:

Subir Archivos

  • En el formulario de contratos:
    • Haz clic en "Seleccionar archivos"
    • Elige uno o más archivos
    • Los archivos se guardarán automáticamente
  • En el formulario de HES:
    • Haz clic en "Seleccionar anexos"
    • Elige uno o más archivos
    • Los anexos se guardarán automáticamente

Almacenamiento

Los archivos se almacenan en la base de datos local (IndexedDB) con la siguiente información:

  • Nombre original del archivo
  • Tipo MIME
  • Tamaño en bytes
  • Fecha de última modificación
  • Contenido en formato base64

Descargar Archivos

Para descargar archivos adjuntos:

  • En la lista de archivos adjuntos, haz clic en el botón de descarga junto al archivo deseado
  • El archivo se descargará con su nombre original
  • Se muestra el tamaño de cada archivo en KB

Limitaciones

  • Los archivos se almacenan localmente en el navegador
  • Hay un límite de tamaño según el navegador (recomendado menos de 5MB por archivo)
  • Los archivos no se sincronizan entre dispositivos
  • Se recomienda hacer copias de seguridad de los archivos importantes

Eliminación

Los archivos adjuntos se eliminan automáticamente cuando:

  • Se elimina el contrato asociado
  • Se elimina la HES asociada
  • Se limpia el formulario

Diseño Responsivo

El diseño de SIGESCON está optimizado para funcionar tanto en escritorio como en dispositivos móviles:

  • Barra de navegación colapsa en pantallas pequeñas
  • Botones ajustables para móvil
  • Texto y elementos reorganizados para mejor legibilidad

Media Queries Personalizadas

Se usan media queries para mejorar la experiencia en diferentes tamaños de pantalla:

@media (max-width: 768px) {
    header {
        flex-direction: column;
        align-items: flex-start;
    }
    .primary-btn, .success-btn, .danger-btn {
        width: 100%;
        justify-content: center;
    }
}
