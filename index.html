Ayuda - SIGESCON

Bienvenido a la sección de ayuda de SIGESCON. Aquí encontrarás información detallada sobre el funcionamiento del sistema.

¿Qué es SIGESCON?

SIGESCON es una aplicación web desarrollada en HTML + JavaScript (con Chart.js y Bootstrap) diseñada para gestionar contratos empresariales, Hojas de Entrada de Servicios (HES), avances físicos, informes y estadísticas.

  • Tecnología: Frontend puro (sin backend)
  • Almacenamiento: Base de datos local con LocalStorage del navegador
  • Exportaciones: Excel y PDF
  • Gráficos: Chart.js
Barra de Navegación

La barra lateral contiene todas las funcionalidades del sistema, divididas en módulos independientes:

  • Resumen General: Muestra estadísticas claves del sistema.
  • Nuevo/Editar Contrato: Formulario para crear o actualizar contratos.
  • Lista de Contratos: Consulta, filtra y exporta tus contratos.
  • Gestión de HES: Registra Hojas de Entrada de Servicios asociadas a contratos.
  • Resumen Gráfico: Visualización gráfica de los datos.
  • Papelera de Reciclaje: Gestiona elementos eliminados.
Módulo: Nuevo/Editar Contrato

Este formulario permite registrar nuevos contratos o editar uno ya existente. Contiene campos como:

  • Número de SICAC
  • Número de Referencia
  • Objeto Contractual / Descripción
  • Contratista
  • Fechas de Inicio y Fin (Duración calculada automáticamente)
  • Monto Original
  • Modalidad de Contratación (dinámica con botones + y -)
  • Partidas (agregables dinámicamente)
  • Estatus
  • Observaciones

Al guardar, el contrato se añadirá a la lista de contratos y sus datos se guardarán en tu navegador.

Módulo: Gestión de HES

Permite registrar Hojas de Entrada de Servicios (HES) para cada contrato. Cada HES debe estar asociada a un contrato existente y contiene sus propias partidas.

  • Número de HES (único por contrato)
  • Fecha de HES
  • Monto Total de HES (calculado automáticamente)
  • Partidas de la HES (referenciando partidas del contrato)
  • Observaciones

Las HES contribuyen al cálculo del avance financiero del contrato.
