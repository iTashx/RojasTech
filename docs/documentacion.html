Introducción

SIGESCON v4.0 es un sistema integral para la gestión de contratos empresariales, preparado para integración con backend y empaquetado con Electron.

Módulos y Arquitectura

  • Gestión de Contratos
  • Partidas y Gasto Administrativo
  • Extensiones
  • HES
  • Avance Físico y Financiero
  • Notificaciones
  • Informes y Reportes
  • Papelera y Recuperación
  • Configuración y Catálogos

Arquitectura modular, cada bloque funcional tiene su propio archivo JS y CSS.

Flujos y Reglas de Negocio

  1. Crear/editar contrato → Añadir partidas y GA → Registrar HES → Registrar extensiones → Consultar avances e informes → Exportar/Importar datos → Gestionar papelera.

Validaciones cruzadas, feedback inmediato y notificaciones centralizadas.

Base de Datos y Persistencia

Soporte para IndexedDB/localStorage y preparado para integración con bases de datos reales (SQLite, MySQL, PostgreSQL, etc.).

  • Contratos, partidas, HES, extensiones, notificaciones, papelera.

Integración Backend/Electron

  • API REST para sincronización de datos
  • Empaquetado con Electron para escritorio
  • Backup/restore y multiusuario (planificado)

Recomendaciones de Uso

  • Probar cada módulo tras una actualización
  • Revisar notificaciones y feedback visual
  • Consultar la ayuda y documentación regularmente
