Documentación SIGESCON

Introducción

SIGESCON es un sistema de gestión de contratos empresariales que permite el seguimiento y control de contratos, HES (Hojas de Estimación de Servicios) y sus respectivas partidas.

Estructura del Sistema

Arquitectura

El sistema está diseñado con una arquitectura modular y escalable:

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Base de Datos:
    • IndexedDB (Dexie.js) para almacenamiento local
    • Preparado para migración a base de datos física (PostgreSQL/MySQL)
  • Almacenamiento:
    • Archivos locales para documentos adjuntos
    • Preparado para migración a almacenamiento en la nube

Migración a Base de Datos Física

El sistema está preparado para migrar a una base de datos física con las siguientes consideraciones:

  • Mantenimiento de la misma estructura de tablas
  • Adición de índices y claves foráneas
  • Implementación de triggers para auditoría
  • Soporte para múltiples usuarios concurrentes

Módulos del Sistema

Gestión de Contratos

  • Creación y edición de contratos
  • Gestión de partidas
  • Seguimiento de vencimientos
  • Alertas configurables

Gestión de HES

  • Creación y edición de HES
  • Cálculo automático de totales
  • Gastos administrativos configurables
  • Seguimiento de ejecución

Reportes y Análisis

  • Exportación a Excel y PDF
  • Gráficos de avance
  • Resúmenes financieros

Configuración del Sistema

Parámetros Configurables

  • Días de Alerta: Configuración del número de días antes del vencimiento para mostrar alertas (1-365 días)
  • Gastos Administrativos: Porcentaje configurable para cada HES
  • Moneda: Selección de moneda para contratos (USD, EUR, VES)

Exportación/Importación

  • Exportación completa de la base de datos
  • Importación de datos desde archivos Excel
  • Respaldo automático de configuración
