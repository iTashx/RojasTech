ESPECIFICACIONES GENERALES
-------------------------
- Formato: PNG o JPG (ambos aceptados)
- Resolución: 1920x1080 o similar
- Tamaño máximo: 1MB por imagen
- Calidad: Alta, con buena legibilidad
- Tema: Claro, para mejor visibilidad
- Nombres de archivo: Todo en minúsculas, sin espacios
- Extensión: .png o .jpg (ambas aceptadas)

NOTAS IMPORTANTES
----------------
1. Todas las imágenes deben ser capturas de pantalla claras y nítidas
2. Evitar información sensible en las capturas
3. Usar un tema claro para mejor legibilidad
4. Asegurar que los textos sean legibles
5. Mantener consistencia en el estilo visual
6. Optimizar las imágenes para web
7. Se aceptan tanto PNG como JPG, usa el formato que mejor se adapte a tus capturas
8. Para el carrusel, asegúrate de que las imágenes relacionadas tengan el mismo tamaño y formato
